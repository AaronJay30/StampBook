# Changelog

## v1.0.0 (2026-03-22)

- Initial public release of StampBook.
- Features:
  - Calendar-based stamp book with monthly pages.
  - Upload, edit, and position stamp images for each day.
  - Responsive design with enforced landscape mode on mobile/tablet.
  - Modal viewer for stamps with delete and loading indicators.
  - Zoom controls in the stamp editor.
  - Custom email confirmation template.
  - Supabase/PostgreSQL backend with constraints and migrations.
  - Error handling and user feedback for uploads and deletions.
  - Modern UI with Tailwind CSS and Next.js 16+ (App Router).
